## About

A minimal, single-page portfolio built as a static site for GitHub Pages. Features a modern dark theme with smooth scrolling sections and responsive design.

## Sections

- **Hero** - Introduction with social links
- **About** - Professional summary and career focus
- **Experience** - Detailed work history from 2012 to present
- **Education** - Academic background
- **Gallery** - Professional experiences and team moments with category filtering

## Tech Stack

- React (compiled to static HTML)
- Tailwind CSS for styling
- Lucide React for icons
- GitHub Pages for hosting

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/varunrao1991/varunrao1991.github.io.git
cd varunrao1991.github.io
```

2. Open `index.html` in your browser

3. Make changes and commit:
```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

## Adding Images

Place your images in the `images/` directory following this structure:
- `images/portrait.webp` - Your profile picture
- `images/signature.webp` - Your signature
- `images/experience/` - Gallery images (use original filenames from data)
- Company logos in root `images/` folder

## Custom Domain Setup

To use with your custom domain:

1. Add a `CNAME` file with your domain name
2. Configure DNS A records pointing to GitHub Pages IPs
3. Enable HTTPS in repository settings

See [GitHub Pages documentation](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site) for details.
